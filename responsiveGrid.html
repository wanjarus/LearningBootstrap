The responsive Bootstrap Grid


Break Points

Bootstrap is based on a 12 column grid. As you can see, if you use more than 12, then it will wrap.

  • Large screens (1200px): wide screen desktop
  • Medium screens (1024px): older square desktops
  • Small screens (768px): tablets
  • Extra small screens (320px): smartphones
